# Basics - Fundamental Examples

This section contains basic examples to help you get started with NLP2CMD.

## Categories

### [app2schema](./app2schema/)
Learn how to convert applications to schema definitions and use them with NLP2CMD.

### [docker_basics](./docker_basics/)
Basic Docker command examples and file repair operations.

### [kubernetes_basics](./kubernetes_basics/)
Fundamental Kubernetes command examples.

### [shell_fundamentals](./shell_fundamentals/)
Core shell command examples, environment analysis, and schema caching.

### [sql_basics](./sql_basics/)
Basic SQL command generation and LLM integration examples.

## Quick Start

Pick any category and navigate to its folder to see specific examples:

```bash
cd 01_basics/[category]
python example.py
```
