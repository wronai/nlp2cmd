# Debugging Tools and Examples

Utilities for debugging and improving shell commands.

## Files

- **generator.py** - Debug command generator
- **intents.py** - Intent debugging utilities
- **keywords.py** - Keyword analysis tools
- **simple_demo.py** - Simple debugging demonstrations
- **test_improvements.py** - Test shell improvements
- **commands_demo.sh** - Shell command demonstrations
- **validation.py** - Command validation tools
- **validation_report.txt** - Validation results report

## Running Examples

```bash
# Generate debug commands
python generator.py

# Debug intent recognition
python intents.py

# Analyze keywords
python keywords.py

# Simple debugging demo
python simple_demo.py

# Test shell improvements
python test_improvements.py

# Run command demos
bash commands_demo.sh

# Validate commands
python validation.py
```

## Purpose

These tools help:
- Identify and fix command issues
- Improve command recognition
- Validate command correctness
- Test shell improvements
- Analyze debugging patterns
