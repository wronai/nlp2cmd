# Bioinformatics Examples

Applications of NLP2CMD in bioinformatics and computational biology.

## Files

- **example.py** - Basic bioinformatics command generation
- **complete_examples.py** - Comprehensive bioinformatics workflows

## Running Examples

```bash
# Basic bioinformatics commands
python example.py

# Complete workflow examples
python complete_examples.py
```

## Use Cases Covered

- DNA/RNA sequence analysis commands
- Protein structure analysis
- Bioinformatics pipeline automation
- Data processing for genomics
